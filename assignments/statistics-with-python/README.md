# 📘 Assignment: Statistics with Python

## 🎯 Objective

Use Python, pandas, and NumPy to explore a dataset and perform core statistical calculations. By the end, you will summarize data clearly and explain what your results mean.

## 📝 Tasks

### 🛠️	Load and Explore a Dataset

#### Description
Write a Python program that loads a dataset with pandas, inspects its structure, and reports useful summary information for each numeric column.

#### Requirements
Completed program should:

- Load a dataset into a pandas DataFrame (you may create a small sample dataset if needed).
- Display the number of rows and columns.
- Show column names and data types.
- Print summary statistics using `describe()` for numeric columns.


### 🛠️	Calculate Statistics with NumPy

#### Description
Choose at least one numeric column from your dataset and calculate key statistics with NumPy. Then compare your results with pandas output to check accuracy.

#### Requirements
Completed program should:

- Compute mean, median, and standard deviation using NumPy.
- Compute minimum, maximum, and at least one percentile (for example, 25th or 75th percentile).
- Print results in a readable format with labels.
- Include a short written interpretation (2-4 sentences) describing what the statistics reveal about the data.
